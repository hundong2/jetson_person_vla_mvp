# 안전 및 개인정보 가이드

사람을 자동으로 감지하거나 식별하는 프로젝트는 기술보다 운영 규칙이 먼저 정리되어야 합니다.

## 이 MVP의 범위

- 얼굴 인식이 아닙니다.
- 법적 신원 확인이 아닙니다.
- 한 세션에서 사용자가 직접 `e` 키로 등록한 대상의 옷 색/외형을 비교하는 학습용 예제입니다.

## 실제 프로젝트로 확장할 때 필요한 것

- 촬영 대상의 명시적 동의
- 녹화 여부와 목적 고지
- 프로필 저장 기간과 삭제 방법
- 저장 데이터 암호화
- 접근 권한 관리
- 오탐/미탐 대응 절차
- 법무/보안 검토

## 권장 설계

1. 기본값은 저장하지 않기입니다.
2. 등록 프로필은 자동 만료되게 만듭니다.
3. 영상 원본 대신 최소한의 이벤트 로그만 남깁니다.
4. 식별 결과는 항상 confidence와 함께 표시합니다.
5. 사람에게 직접 영향을 주는 액션은 사람 승인 단계를 둡니다.
